Temporal Cluster

To study how the vector field influences the target, the vector fields are clustered in time. If during a period of time the vector patterns remain almost the same, the algorithm classifies them as one group.

Spatial Cluster

The distance between two streamlines is defined initially. When the distance is within the predefined value, the closet streamlines are clustered together into one cluster.

Line Integral Convolution

LIC employs a low-pass filter to convolve an input noise texture along pixel-centered symmetrically bi-directional streamlines to exploit spatial correlation in the flow direction.

Parallel Coordinate

This is realized through D3.js, it is a tool of visualizing high-dimensional geometry and analyzing multivariate data.

Classification With SVM
